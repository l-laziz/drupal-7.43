
-- SUMMARY --

The User Track module allows to track the registered users login to the site.
The parameters like IP Address, Agent, Access time and Timezone of the access
to the site are tracked. All these tracking details can be viewed by the admin
or the user with given permission to track the user login.


-- REQUIREMENTS --

None.


-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.
